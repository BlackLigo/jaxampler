# Jaxampler

Jaxampler is a tool for sampling statistical distributions using the [JAX](https://jax.readthedocs.io/en/latest/) library.

## Samplers

-   [x] Inverse Transform Sampling
-   [ ] Rejection Sampling
-   [ ] Metropolis-Hastings
-   [ ] Hamiltonian Monte Carlo
-   [ ] Slice Sampling
-   [ ] Gibbs Sampling
-   [ ] Importance Sampling
